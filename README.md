🛍️ Anazon - Modern E-Commerce Platform

Live Demo: https://anazon.vercel.app/

📘 Overview

Anazon is a full-featured e-commerce platform built to showcase modern web development best practices.
Inspired by leading online retailers, it offers a seamless shopping experience with features like product browsing, cart management, wishlists, order tracking, and secure payments.

The project emphasizes performance, SEO, and security, making it a great reference for developers learning to build scalable web applications with Next.js.

✨ Features

🔐 User Authentication – Secure login and session handling via NextAuth (Auth.js).

🛒 Product Catalog – Browse, filter, and search products with personalized recommendations.

🧺 Shopping Cart – Add/remove items and view summaries with real-time price calculations.

💖 Wishlist – Save favorite items for future purchase.

📦 Order Management – Track order status, view history, and reorder easily.

💳 Payment Integration – Secure checkout experience (Stripe or similar).

📱 Responsive Design – Fully optimized for mobile, tablet, and desktop.

⚙️ Error Handling – Custom 404 and user-friendly error pages.

🔍 SEO Optimization – Server-side rendering (SSR) for improved search visibility.

🧰 Tech Stack
Category	Technology
